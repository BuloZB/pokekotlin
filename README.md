# PokeKotlin

Kotlin wrapper for [PokeApi](https://github.com/phalt/pokeapi). Currently under development.

## Status

Model classes: 100%

API interface: 100%

Testing: about 60%

## Usage

### Gradle

```groovy
dependencies {
    compile 'me.sargunvohra.lib:PokeKotlin:<version>'
}
```

### Kobalt
```kotlin
dependencies {
    compile("me.sargunvohra.lib:PokeKotlin:<version>")
}
```
